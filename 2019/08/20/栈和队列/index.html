栈和队列

发表于 2019-08-20 | 分类于 数据结构 |

栈

限定在表尾（栈顶）插入或删除操作的线性表；
后进先出LIFO

顺序栈

栈的抽象数据类型定义：

利用一组地址连续的存储单元存放数据；
先为栈分配基本容量，当空间不够时再分段扩展
需要两个指针：base指向栈底元素，top指向栈顶元素的下一个位置

栈的应用

1 数制转换

对于输入的任意一个非负十进制整数，打印输出与其等值的k进制数

计算原理：$n = (n / k) * k + n mod k$

计算过程是从低位到高位，而打印时正好相反，故可以利用栈后进先出的特性来存储并输出

2 括号匹配

假设表达式中允许包括两种括号——圆括号和方括号，输入括号序列，判断序列中的括号是否匹配

4 表达式求值

根据逆波兰表示法，求表达式的值

leetcode

示例

输入: [“2”, “1”, “+”, “3”, “*”]
输出: 9
解释: ((2 + 1) * 3) = 9

队列

一种先进先出FIFO的线性表,只允许在表的一段插入、另一端删除

链队列

循环队列

当rear == front时，队列的状态可能是空也可能是满，为了解决这种冲突有两种处理方法：

  • 另外设置一个标签位来区别队列是空还是满
  • 少用一个空间，让头指针front指向队头元素，尾指针rear指向队尾元素的下一个元素

采用第二种方法的判定条件为：

  • 队列为空：front == rear
  • 队列满：(rear + 1) % capacity == front
