102 二叉树的层次遍历

link

题目

给定一个二叉树，返回其按层次遍历的节点值。 （即逐层地，从左到右访问所有节点）。

思路

这里使用宽度优先搜索

利用队列实现

  1. 将根节点加入队列（注意处理根节点为空的情况）
  2. 新建数组v（用来保存当前层的节点）
  3. 遍历当前队列的所有节点，将当前节点的后继节点加入队列，将当前节点出队并加入数组v
  4. 将数组v加入容器res
  5. 重复2到4的步骤，直到队列为空

递归实现

  1. 用一个全局变量levels来存储结果
  2. level表示当前的层级（从0开始），当level == levels.size()时，需要新建当前层级的容器v，并将空的v加入levels;
  3. 将当前节点的值加入v
  4. 递归处理左右节点, helper(node->left/right, level + 1)
