C++中的函数

发表于 2019-09-27 |

形参和实参

  • 形参parameter： 函数定义中函数名字后面的圆括号内的是形参列表
  • 实参argument: 调用函数时函数名后的圆括号内的时实参列表
  • 实参是形参的初始值

参数传递

值传递

值传递会将实参拷贝一份副本作为局部变量传进函数，在函数内修改该副本不会改变原来实参的值

指针传递

指针传递本质上也是值传递，只是传递的值是指针变量存储的地址值。修改该地址值指向的变量可以修改变量的值
传递一个数组时，实际上传递的是指向数组首元素的指针。

引用传递

形参相当于实参的别名，对形参的操作就是对实参的操作。
不能将const对象，字面值或者需要类型转换的对象传递给普通的引用形参

  • 使用引用传递可以避免拷贝
  • 使用引用传参返回额外信息

可变形参的函数

C++11中提供了两种方法：

  1. initializer_list
  2. 可变参数模板
    具体可以看这篇文章

返回类型

返回的值将被拷贝到调用点。

  • 不要返回局部对象的引用或指针。 函数完成后它所占用的存储空间将被释放，局部引用和指针会失效
  • C++ 新标准规定函数可以返回花括号包围的值的列表。
  • 因为数组不能被拷贝所以不能返回数组，但可以返回数组指针

  1. 使用类型别名

  2. type (* fun(parameter_list)) [dimension]

  3. 使用尾置返回类型
    尾置返回类型跟在形参列表后面并且以->符号开头。本应该出现返回类型的位置放置auto
    auto fun(int i) ->int(*)[10];

注意区分指针数组和数组指针

  1. 使用decltype

函数重载

同一个作用域内，几个函数名字相同但形参列表不同，称为重载函数

  • main函数不能重载
  • const_cast常用于重载函数的情景中

函数指针

函数的类型由它的返回类型和形参类型决定。
例如函数int f(int i)的类型是int(int)。
如果想声明一个指向该函数的指针，只需将函数名替换成指针：int (*p)(int)。

当把函数名作为一个值使用时，该函数自动转换成指针。

可以直接调用函数指针，不需要解引用
